# Ecommerce Website

##  About the Project
A dynamic e-commerce website built as a school project using PHP, MySQL, HTML, CSS, and JavaScript. This website includes user login, product display, and integrates a database for product management.

##  Features
- User authentication (login, register)
- Products display and browsing
- Database integration with MySQL
- Custom CSS and JavaScript for UI improvements

##  Project Structure
- index.php – Home page
- login.php / register.php – Authentication
- css/ – Styles files
- js/ – JavaScript files
- includes/ – PHP includes (like database connection)
- database.sql – Database schema file

## Technologies Used
- *PHP*
- *HTML5*
- *CSS3*
- *JavaScript*
- *MySQL*

##  How to run the project
1. Install XAMPP or WAMP  
2. Put project folder in htdocs (for XAMPP)  
3. Open phpMyAdmin  
4. Create a database and import database.sql  
5. Run the website: http://localhost/your-folder
